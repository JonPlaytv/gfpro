F5-TTS Voice Cloning Integration
Replace the browser's Web Speech API with F5-TTS, a state-of-the-art local AI TTS model, to give your AI partner a completely custom, cloned voice. You only need to provide a voice sample once — the app will use it for every response after.

How F5-TTS Voice Cloning Works
F5-TTS doesn't need to "train" a separate model. It uses zero-shot voice cloning via Flow Matching:

You record ~10 seconds of any voice (your own, an anime character's, anything).
On every TTS request, F5-TTS receives: (a) your reference audio clip, (b) the transcript of that clip (what was said in it), and (c) the new text to generate.
It then generates new audio that sounds exactly like the voice in the reference clip.
That's it — no fine-tuning, no training. The reference clip is the "key" to the voice.

User Review Required
IMPORTANT

GPU Required for Real-Time Performance: F5-TTS is a diffusion model. On an NVIDIA GPU it generates audio in 1–3 seconds. On CPU it can take 30–60 seconds, which would break conversational flow. Please confirm if you have a CUDA GPU available.

IMPORTANT

Python Environment Needed: A Python environment (conda or venv) with f5-tts, fastapi, and uvicorn must be set up. I will provide a setup script and all commands.

NOTE

Voice Sample: You need a clean audio clip (WAV, ~5–15 seconds) of the voice you want to clone. No background noise. You will also need to know the transcript of what was said in it. The app will have a built-in recorder to capture this directly from your browser.
